# Frontend for Gen-Archi

## Build Docker Image
```bash
docker build -t sample:dev .
```

## Launch Docker Container
```bash
docker run \
    -it \
    --rm \
    -v ${PWD}:/app \
    -v /app/node_modules \
    -p 3001:3000 \
    -e CHOKIDAR_USEPOLLING=true \
    sample:dev
```
